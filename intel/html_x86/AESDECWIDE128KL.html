Using 128-Bit Key
Instruction Operand Encoding
Description
The AESDECWIDE128KL
1
instruction performs ten rounds of AES to decrypt each of the eight blocks in XMM0-7
using the 128-bit key indicated by the handle from the second operand. It replaces each input bloc k i n X M M 0- 7 w i t h
its corresponding decrypted block if the operation succeeds (e.g., does not run into a handle violation failure).

Flags Affected
ZF is set to 0 if the operation succeeded and set to 1 if the operation failed due to a handle violation. The other
arithmetic flags (OF, SF, AF, PF, CF) are cleared to 0.
